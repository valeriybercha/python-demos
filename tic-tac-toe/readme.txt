
========= TIC-TAC-TOE game ==========

Description: TIC-TAC-TOE game written 
on Python for 2 players to play.

A player should enter available number
from the grid to mark the grid place.

Game grid:

1 | 2 | 3
---------
4 | 5 | 6
---------
7 | 8 | 9

Input commands:
- "info", "i" - readme information
- "quit", "q", "end", "exit" - quit the progra
