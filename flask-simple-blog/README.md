# Basic demo blog created with Flask

### Basic information:

- Blog app created with Flask and Python
- Programming language: Python 3.6.9
- Developer: Valeriy B.

### Blog functionality:

- Add a post
- Add a page
- Update a post
- Delete a post
- Update a page
- Delete a page

### Libraries to install:
- ```pip install flask```
- ```pip install flask-sqlalchemy```
    
This app is created for showcase purposes only
